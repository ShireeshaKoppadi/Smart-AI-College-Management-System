# 🤖 Smart AI College Management System

An AI-powered College Management System developed using **Python Flask, HTML, CSS, JavaScript, SQLite, and Machine Learning**.

This system helps colleges manage student information, attendance, and academic performance through a smart digital platform.

---

## 🚀 Features

### 👨‍🎓 Student Management
- Store and manage student details
- Maintain academic records
- Easy student information access

### 📅 Smart Attendance System
- Track student attendance
- Generate attendance reports
- Monitor attendance percentage

### 🧠 AI Performance Prediction
- Predict student academic performance
- Analyze student data using Machine Learning

### 📊 Dashboard
- View important college statistics
- Smart and user-friendly interface
